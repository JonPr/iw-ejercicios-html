    ¿Quiénes somos?

    Procesados de patata Paturpat S. Coop. nace en el año 2016 de la mano de Udapa con la intención de dar respuesta a las nuevas tendencias de consumo demandadas por los consumidores a través de productos fáciles de preparar, nutritivos y saludables. Tras una intensa labor de investigación con personal propio y el apoyo de centros tecnológicos de referencia en el sector agroalimentario, Paturpat ha desarrollado productos listos para su consumo dentro de la denominada 5ª gama.

    Tipos de productos

    Disponemos de tres tipos de productos con distintos tipos de cortes y/o elaboraciones:

    • Patata de 5ª Gama

      En este lote de productos nos podemos encontrar diferentes tipos de corte de patata orientados a determinados platos.

      • Patata para tortilla
      • Patata para guisos y tapas
      • Patata para guarnición
      • Patata para ensaladilla

    • Patatas con salsas

      Se tratan de patatas ya preparadas para meterlas unos minutos en el microondas y tenerlas listas para comer.

      • Patata para microondas con salsa mojo picón
      • Patata para microondas con salsa Alioli
      • Patata para microoondas con crema normanda
      • Patata para microondas con salsa brava

    • Patata para industria y canal Horeca

      Se trata de una pata principalmente destinada a la industria.

      • Patata parisina
      • Patata para guisos y tapas
      • Patata para guarnición
      • Patata para ensaladilla
      • Patata para tortilla
      • Bastones para frito
